# v0.6 Report Polish and Optional PDF Export

v0.6 improves the reporting layer without changing the causal estimation core.

## Highlights

- Polished HTML report layout with summary cards.
- Print-friendly HTML CSS for browser print-to-PDF workflows.
- Optional PDF export powered by ReportLab.
- Graceful fallback when ReportLab is not installed.

## Dependency Boundary

PDF export is optional and uses `requirements-pdf.txt`:

```powershell
.\.venv\Scripts\python.exe -m pip install -r requirements-pdf.txt
```

ReportLab is not part of the base requirements. Missing ReportLab does not affect Markdown export, HTML export, Data Quality Checks, LangGraph fallback, ATE, CATE, refutation, or Reviewer behavior.

## Non-Goals

- No causal estimation changes.
- No database, login, deployment, OpenAI API, checkpoint, persistence, human-in-the-loop, LLM planner, or dynamic routing.
- No real business data handling.
